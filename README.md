#Об исследовании

Экологические, социальные и санитарные характеристики общетсвенных просстранств зачастую рассматриваются раздельно — это не позволяет выявлять их взаимосвязь и обосновывать решения по реконструкции с помощью одного инструмента.

Цель работы — разработать метод комплексной оценки общественных пространств для соблюдения экологического баланса и социальных целей и обоснования решений по реконструкции.

Объект: локальные открытые общественные места, включенные в повседневное посещение соседскими сообществами.

Предмет: методы и процедуры оценки, позволяющие выявлять баланс территории.

Метод: три тематических индекса
Каждый параметр измеряется по шкале от 1 до 5 баллов.
# ЭКО — Экологический индекс 
Интегральная оценка экологического качества и микроклиматического комфорта зоны. Параметры: озеленение (NDVI), теневое покрытие, водопроницаемость и дренаж.
# СОЦ — Индекс социальной насыщенности 
Интегральная характеристика разнообразия и выраженности влияет на использование и взаимодействие возможностей. Параметры: поведенческие сценарии, возрастная представленность, временная активность.
# САН — Индекс состояния здоровья территории 
Интегральная оценка состояния и «ухоженности» окружающей среды. Параметры: чистота покрытия, состояние элементов, признаки вандализма.
# Баланс ЭКО + СОЦ + САН определяется на уровне территории функционально-пространственных зон

Апробация: три объекта Санкт-Петербурга

# Область применения
Метод предназначен для предпроектной стадии анализа и может использоваться проектными и исследовательскими командами, обеспечивающими управление и заказчиками благоустройства для учета дефицитов, приоритета вмешательства и обоснования проектных решений.
